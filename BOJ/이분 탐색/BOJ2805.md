# 백준 2805번
[백준 2805번](https://www.acmicpc.net/problem/2805)
```java
import java.util.*;

public class Main{
	
	public static void main(String[] args) {
		Scanner kb = new Scanner(System.in);
		
		int N = kb.nextInt();
		int M = kb.nextInt();
		int arr[] = new int[N];
		
		for(int i = 0; i < N; i++) {
			arr[i] = kb.nextInt();
		}
		Arrays.sort(arr);
		long max = arr[N-1];
		
		long lt = 1;
		long rt = max;
		
		while(lt <= rt) {
			long mid = (rt+lt)/2;
			long length = 0;
			
			for(long x : arr) {
				if(x>=mid) {
					length += (x-mid);
				}
			}
			
			if(M <= length) {
				lt = mid + 1;
			}else {
				rt= mid -1;
			}
			
		}
		System.out.print(rt);
	
	}
}
```
- 1654번 랜선자르기 문제와 유사하다.
- 랜선 자르기는 자르는 횟수를 구하기 위해 몫을 구했고 나무 자르기는 길이를 구하기 위해 각 원소마다 mid값을 빼준다.
- 길이에 음수가 더해지는 것을 막기 위해 x가 mid보다 같거나 크게 설정해준다.
- 자료형을 주의하자!
